## ⚙️ How It Works

### 🧩 Product System
Products are stored in a JavaScript array and dynamically rendered into the DOM.

- Each product contains:
  - ID
  - Name
  - Price
  - Category
  - Image
  - Description

Products are displayed using DOM manipulation and updated instantly when filters are applied.

---

### 🛒 Cart System
The cart uses an array-based structure:

- Items are stored with quantity tracking
- Total price updates automatically
- Cart is saved in **localStorage**
- Items persist across sessions

---

### 🔎 Search & Filtering
- Debounced search input for performance
- Filters products by:
  - Name
  - Description
- Works together with category filters
- Handles empty results gracefully

---

### ✅ Form Validation
- Email format validation
- Required field validation
- Phone number checks
- Address completeness check
- Terms & conditions required before checkout
